TIPS FOR BEST RESULTS:
================================================================================

1. MIDJOURNEY USERS: Add these parameters
   - "--ar 16:9" for hero images
   - "--ar 4:3" for products/projects
   - "--ar 1:1" for square images
   - "--style raw" for photorealistic results
   - "--v 6" for latest version

2. DALL-E 3 USERS:
   - Use "photorealistic" and "8k resolution" in prompts
   - Specify "architectural photography" for professional look
   - Request "natural lighting" for outdoor scenes

3. IDEOGRAM USERS:
   - Select "Photo" mode
   - Use "Magic Prompt" for enhancement
   - Choose 16:9 aspect ratio for heroes

4. GENERAL TIPS:
   - Generate multiple variations and pick the best
   - Avoid text/logos in images (AI struggles with text)
   - For consistency, use similar lighting descriptions
   - Download in highest resolution available
   - Optimize images before uploading (TinyPNG, Squoosh)

5. INDIAN CONTEXT:
   - Mention "Indian architecture" or "India" for authentic look
   - Reference specific cities for local flavor
   - Include tropical/monsoon elements where relevant
